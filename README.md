# PSoC-Keyboard-to-LED-Control-Project
# Description 

Ce projet consiste à développer un programme embarqué sur microcontrôleur PSoC permettant de contrôler une ou plusieurs LEDs à travers un clavier matriciel.

Le projet inclut :

-Détection des touches via lecture des registres

-Manipulation des registres PRTXDR

-Configuration des résistances de tirage internes

-Gestion logique des entrées/sorties

-Contrôle multiple de LEDs

# Objectifs techniques

*Déterminer les adresses hexadécimales des touches
et Configurer le module LED

*Paramétrer le brochage du microcontrôleur

*Commander les LEDs en fonction des touches
# Concepts utilisés
Registre PRTXDR
/Logique AND (&)
/Gestion des ports numériques
/Pull-up / Pull-down resistors
/Interaction hardware/software

# Compétences démontrées

-Programmation C embarquée

-Accès direct aux registres

-Configuration hardware

-Conception logique digitale

# Environnement et logiciel

-Microcontrôleur PSoC

-Carte de développement avec LEDs D9-D16

-psoc design
